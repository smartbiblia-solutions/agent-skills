## Commentaires

La zone 104 du format UNIMARC-Sudoc correspond à plusieurs positions de la zone 100 dans le format UNIMARC standard. On la complète aisément grâce aux grilles de données codées (mode de catalogage " semi-expert ").

**Ressources continues**

La zone 104 étant une zone sous autorité de l'ISSN, avant toute modification se référer à la [procédure de demande de correction](https://documentation.abes.fr/sudoc/manuels/controle_bibliographique/circuit_signalement_rc/index.html#ProcedureDemandeCorrectionISSN ), puis effectuer, le cas échéant, une demande de correction dans [CIDEMIS](https://cidemis.sudoc.fr/)..

Les seules sous-zones à considérer sont les sous-zones **$d** (écriture du titre) et **$f** (langue de catalogage = fre, pour toute notice produite dans Sudoc). Les autres sont supprimées par la mise à jour ISSN.

### Sous-zone 104 $e : transcription des données

La valeur habituelle est **0 = transcription fidèle**.




Le code **1 = transcription appauvrie** (traduit par " notice modifiée " dans le *Manuel UNIMARC : format bibliographique* 4e éd. page 72, pour la zone UNIMARC 100 pos.21) doit être utilisé pour les notices translittérées ou celles dans lesquelles certains caractères du document ne sont pas reproduits (on ne tient pas compte des symboles ou dessins qui n'existent dans aucun jeu de caractères).
