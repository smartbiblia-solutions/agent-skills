## Commentaires

NB : Pour le traitement des mentions de responsabilité comportant plus de trois noms de personne ou de collectivité voir la [fiche dédiée](https://documentation.abes.fr/sudoc/regles/Catalogage/Regles_PlusDe3Auteurs.htm)

L'une des sous-zones **$a** ou **$3** est obligatoire.

Le **$3**[ppn de la notice autorité liée] est exclusif des sous-zones alphabétiques.

Lorsqu'un lien est établi vers une autre notice, la sous-zone **$3** *remplace* ces sous-zones.

Après validation de la notice, l'interface de catalogage du Sudoc affiche automatiquement le contenu des sous-zones alphabétiques de la notice autorité liée par le **$3** (c'est ce qu'on appelle l'"expansion" du lien).

L'emploi d'une zone 700 exclut la présence d'une zone 710 ou 720 dans la même notice. Les autres mentions de responsabilités principales doivent faire l'objet de points d'accès 7X1 (701 / 711 / 721).

### $4 Code de fonction

Dans le Sudoc, un code de fonction **$4** est **obligatoire** dans toutes les zones 7XX des notices bibliographiques. Voir [Enregistrement des points d'accès auteur à partir d'octobre 2019](https://documentation.abes.fr/sudoc/regles/Catalogage/index.htm#ChoixAcces).

La sous-zone **$4** est répétable.

Il est possible de saisir plusieurs fonctions dans plusieurs **$4** d'une même 700 (ou [701](https://documentation.abes.fr/sudoc/formats/unmb/zones/701.htm)), de même que plusieurs fonctions dans plusieurs **$4** d'une même [702](https://documentation.abes.fr/sudoc/formats/unmb/zones/702.htm).

| Exemple : |
| --- |
| **200** 1#**$a**La @Schtroumpfette et le chevalier**$f**Peyo |
| [...] |
| **700** #1**$3**027069427*Peyo (1928-1992)***$a***$4070$4440* |
| Commentaire : |
| La sous-zone **700 $4** répétée permet de signaler que l'auteur de la bande dessinée est à la fois le créateur des textes (070) et des dessins (440) |

| Exemple : |
| --- |
| **200** 1#**$a**Il @bidone**$f**Federico Fellini, réalisateur, scénariste**$g**Ennio Flaiano,<br>Tullio Pinelli, scénaristes**$g**Nino Rota, compositeur**$g**Broderick Crawford, Franco Fabrizi, Giulietta<br>Masina... [et autres], acteurs |
| **700** #1**$3**027291537*Fellini, Federico (1920-1993)**$4300$4690* |
| **701** #1**$3**028467205*Flaiano, Ennio (1910-1972)***$4**690 |
| **701** #1**$3**030201608*Pinelli, Tullio (1908-....)***$4**690 |
| Commentaire : |
| La sous-zone **700 $4** est répétée pour introduire les fonctions cumulées de réalisateur (300) et de scénariste (690) du film |

Attention à ne pas mélanger en revanche des fonctions relevant des entités Oeuvre - Expression (en 700-701) et des fonctions relevant des entités Manifestation - Item derrière une même étiquette 7XX. Dans un tel cas on fait une 700/701 + une 702, chacune dotée des $4 qui leur correspondent.

| Exemple : |
| --- |
| **200** 1#**$a**@Entretiens, fragments et sentences?**$f**Epictète**$g**introduction, traduction par Robert Muller |
| [...] |
| **700** #1**$3**026851822*Épictète (0050?-0130?)***$4**070 |
| **701** #1**$3**029761425*Muller, Robert (1944-....)**$4730* |
| **702** #1**$3**029761425*Muller, Robert (1944-....)**$4080* |
| Commentaire : |

Le code **000 "Fonction à préciser"** est implémenté dans le Sudoc par l'ABES dans toutes les zones 7XX (sauf 716) dépourvues de $4, à partir de fin mars 2013. Il n'est pas affiché dans l'interface publique du catalogue Sudoc, ni exporté dans les notices lors des transferts réguliers.

**Il doit être remplacé en catalogage courant par un code pertinent de la liste des codes de fonction.**La note [309](https://documentation.abes.fr/sudoc/formats/unmb/zones/309.htm) invitant les catalogueurs à corriger ces codes 000 doit être supprimée après correction.
