# Zone 320 : Note sur les bibliographies ou les index contenus dans la ressource

## Descriptif

| Zone | Ind1 | Ind2 | Code de sous-zone | Position | Valeur | O/F | R/NR | Contenu |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 320 | # | # |  |  |  | F | R | Note sur les bibliographies ou les index contenus dans la ressource |
|  |  |  | a |  |  | O | NR | Texte de la note |
|  |  |  | u |  |  | F | NR | URI |
|  |  |  | 2 |  |  | F | NR | Texte du lien hypertexte |

Les informations figurant en couleur dans le tableau ne sont pas telles que définies dans le format Unimarc standard

## Commentaires

Cette note indique l'existence d'une bibliographie (complète ou sous forme de notes) et/ou d'un index dans un document, avec, de façon facultative (mais vivement recommandée dans le Sudoc), leur pagination.

Conformément à RDA-FR, § 3.54.1.4, on doit abandonner l'usage des abréviations.

Concernant la bibliographie : si elle se présente à la fois sous forme complète et sous forme de notes, on indique d'abord la bibliographie (avec pagination), puis les notes bibliographiques (sans pagination, avec éventuellement une précision si elles sont réunies en fin d'ouvrage).

Quand une des pages de la bibliographie n'est pas paginée (alors qu'elle est contenue dans la pagination totale de l'ouvrage), on doit mettre la page "non paginée" entre crochets.
