## Commentaires

NB : Pour le traitement des mentions de responsabilité comportant plus de trois noms de personne ou de collectivité voir la [fiche dédiée](https://documentation.abes.fr/sudoc/regles/Catalogage/Regles_PlusDe3Auteurs.htm)

L'une des sous-zones **$a** ou **$3** est obligatoire.

Le **$3**[ppn de la notice autorité liée] est exclusif des sous-zones alphabétiques.

Lorsqu'un lien est établi vers une autre notice, la sous-zone **$3** *remplace* ces sous-zones.

Après validation de la notice, l'interface de catalogage du Sudoc affiche automatiquement le contenu des sous-zones alphabétiques de la notice autorité liée par le **$3** (c'est ce qu'on appelle l'"expansion" du lien).

### $4 Code de fonction

Dans le Sudoc, un code de fonction **$4** est **obligatoire** dans toutes les zones 7XX des notices bibliographiques. Voir [Enregistrement des points d'accès auteur à partir d'octobre 2019](https://documentation.abes.fr/sudoc/regles/Catalogage/index.htm#ChoixAcces).

La sous-zone **$4** est répétable.

Il est possible de saisir plusieurs fonctions dans plusieurs **$4** d'une même [700](https://documentation.abes.fr/sudoc/formats/unmb/zones/700.htm) (ou [701](https://documentation.abes.fr/sudoc/formats/unmb/zones/701.htm)), de même que plusieurs fonctions dans plusieurs **$4** d'une même 702.

Attention à ne pas mélanger en revanche des fonctions relevant des entités Oeuvre - Expression (en 700-701) et des fonctions relevant des entités Manifestation - Item derrière une même étiquette 7XX. Dans un tel cas on fait une 700/701 + une 702, chacune dotée des $4 qui leur correspondent.

| Exemple : |
| --- |
| **200** 1#**$a**@Entretiens, fragments et sentences**$f**Epictète**$gintroduction, traduction par Robert Muller** |
| [...] |
| **700** #1**$3**026851822*Épictète (0050?-0130?)**$4070* |
| **701** #1**$3**029761425*Muller, Robert (1944-....)**$4730* |
| **702** #1**$3**029761425*Muller, Robert (1944-....)**$4080* |
| Les responsabilités associées à une même personne sont réparties dans 2 zones 7XX distinctes : 700 pour le rôle d'auteur ($4 = 070) et 702 pour le rôle de préfacier ($4 = 080) |

L'usage du code de fonction 920 [Propriétaire actuel] pour enregistrer la propriété juridique du document, interdit l'affichage de la zone dans le Sudoc public.
