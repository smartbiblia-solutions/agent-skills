# Zone 215 : Description matérielle

## Descriptif

| Zone | Ind1 | Ind2 | Code de sous-zone | Position | Valeur | O/F | R/NR | Contenu |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| 215 | # | # |  |  |  | F | R | Description matérielle |
|  |  |  | a |  |  | F | R | Type de présentation matérielle et importance matérielle |
|  |  |  | b |  |  | F | NR | Matériau et technique |
|  |  |  | c |  |  | F | NR | Autres caractéristiques matérielles |
|  |  |  | d |  |  | F | R | Dimensions |
|  |  |  | e |  |  | F | R | Matériel d'accompagnement |

## Commentaires

Se reporter aux [fiches techniques](https://documentation.abes.fr/sudoc/regles/Catalogage/index.htm) et aux [normes](https://documentation.abes.fr/sudoc/normes/normes.htm) pour chaque type de document.


Conformément à RDA-FR, les abréviations ne doivent plus être utilisées (exemples : ill., couv.ill., vol., p.) sauf lorsque l'on retranscrit une information abrégée présente sur le document décrit.

Cependant, l'abréviation "n°" ou "N°" pour "numéro" est acceptée.

Pour les dimensions (voir RDA-FR 3.5) : utiliser les symboles métriques : "mm" pour des millimètres, "cm" pour des centimètres, "m" pour des mètres.

Pour les durées (voir RDA-FR 3.4) : utiliser les abréviations de la norme *ISO 31-1 Grandeurs et unités. parie 1, Espace et temps*, à savoir "h" pour les heures, "min" pour les minutes, "s" pour les secondes.

Dans le cas où le matériel d'accompagnement (signalé en 215 $e) est absent de la ressource (parce qu'il n'a pas été acheté par la bibliothèque ou parce qu'il a été perdu), l'information doit être précisée en note [E316](https://documentation.abes.fr/sudoc/formats/E316.htm). Cette absence ne justifie pas la création d'une nouvelle notice.
