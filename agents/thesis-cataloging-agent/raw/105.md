## Sous-zones

### $b : Nature du contenu

Cette sous-zone, répétable jusqu'à 4 fois, correspond en Unimarc export au 105 $a position 4-7
"Nature du contenu".

*Remarques importantes :*

- La valeur **a** s'attribue à un document *qui est* ou *qui contient* une bibliographie. Il en est de même de toutes les autres valeurs dans cette rubrique, sauf **c** (index), qui signifie que le document *dans son ensemble* est un index (si le document contient un index, voir [$e](https://documentation.abes.fr/sudoc/formats/unmb/zones/105.htm#$e) ci-dessous).
- La 1ère occurrence du **$b** ne doit pas contenir la valeur **#** : pour un type de contenu qui ne figure pas dans la liste, utiliser la valeur **z** (Autres).

*Pour les thèses et travaux universitaires*, se référer à la fiche technique [Catalogage des thèses, chapitre 1.4](https://documentation.abes.fr/sudoc/regles/Catalogage/Regles_Theses.htm#DCMonographies).

- La valeur **m** s'applique aux thèses, document originel (document de soutenance). Dans le Sudoc, cette valeur ne s'applique pas aux mémoires, pour permettre le fonctionnement du filtre de recherche sur les thèses. Lorsqu'il y a plusieurs $b il est IMPERATIF de saisir le $bm en première position.
- La valeur **v** s'applique aux thèses et travaux universitaires, versions publiées, et reproductions.
- La valeur **7** spécifique au Sudoc s'applique aux mémoires, document originel.  NB : Ce dernier code, spécifique au Sudoc permet de filtrer les mémoires lors de la recherche dans le catalogue Sudoc public avec limitation aux thèses.

Une de ces trois valeurs est obligatoire dans le Sudoc pour les notices de travaux universitaires.

### $e: index

On entre dans cette sous-zone la valeur 0 si le document ne contient pas d'index et la valeur 1 s'il contient un index.

Si le document est en lui-même un index, on l'indique en [$b](https://documentation.abes.fr/sudoc/formats/unmb/zones/105.htm#$b).
