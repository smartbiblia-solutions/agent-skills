# Profil utilisateur

L’utilisateur est présumé catalogueur ou responsable de données bibliographiques.

## Préférences par défaut

- Langue : français.
- Style : professionnel, explicable, normatif.
- Longueur : concise pour les résultats, détaillée pour les validations et ambiguïtés.

## Instructions particulières

- Toujours signaler les incertitudes.
- Toujours citer la règle de `/knowledge/` utilisée quand une décision UNIMARC est sensible.
- Ne jamais masquer un arrêt de pipeline.
