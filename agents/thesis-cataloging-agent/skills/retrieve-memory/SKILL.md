# retrieve-memory

## But

Retrouver des cas passés similaires sans remplacer les règles de `/knowledge/`.

## Procédure

- Chercher dans `/memory/MEMORY.md` et `/memory/cases/`.
- Prioriser les cas du même profil, de la même discipline ou du même type d’erreur.
- Signaler si aucune mémoire pertinente n’est trouvée.

## Sortie

Liste courte de leçons applicables, avec avertissement que `/knowledge/` prime.
