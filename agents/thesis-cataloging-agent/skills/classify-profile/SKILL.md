# classify-profile

## But

Classer le document dans `these_originale_imprimee` ou `memoire_original_imprime`.

## Règles

- Accepter seulement les documents imprimés originaux.
- Exclure électronique, reproduction, édition commerciale, HDR, manuscrit, thèse d’exercice et cas incertain selon le profil.
- Exiger une confiance élevée.

## Arrêt

Si le profil est hors périmètre ou incertain :

```text
Le document ne correspond pas avec certitude aux deux profils locaux supportés. Une revue par un catalogueur humain est nécessaire.
```
